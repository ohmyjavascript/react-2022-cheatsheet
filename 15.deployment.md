## Lazy loading 

- leaner bundle.

```
import { Suspense } from 'react';
const NewQuote = React.lazy(() => import('./pages/NewQuote'));

<Suspense fallback={<div>loading..</div>}>
 <Switch>
  <Route>
      <NewQuote />
  </Route>
  </Switch>
</Suspense>
```
